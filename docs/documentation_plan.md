## 4. Troubleshooting Guide Outline (`docs/troubleshooting.md`)

### Sections
1. **Ollama Not Reachable**
   - `ollama serve` not running
   - Wrong `OLLAMA_BASE_URL`
   - Firewall blocking port 11434

2. **Resume PDF Won't Parse**
   - Image-based PDF (needs OCR)
   - File not at `resume/resume.pdf`
   - `pdfplumber` extract_text returns empty

3. **LLM Returns Malformed JSON**
   - Re-run `analyze`
   - Lower temperature if model supports it
   - Check that analyzer_system.txt has the schema example

4. **Tracker File Is Locked**
   - Close Excel before running commands
   - PermissionError on `wb.save()`

5. **URL Fetch Fails**
   - Page requires JavaScript (job boards with dynamic loading)
   - Check `--url` scheme is http/https
   - Timeout after 10 seconds

6. **Streamlit Won't Start**
   - `streamlit` not installed
   - Port 8501 in use
   - Wrong working directory
